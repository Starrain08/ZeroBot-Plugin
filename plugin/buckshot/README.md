# 恶魔轮盘插件

## 代码结构

- `buckshotroulette.go` - 主入口文件，包含初始化和路由注册
- `game.go` - 游戏数据结构和游戏状态管理
- `handlers.go` - 消息处理函数
- `items.go` - 道具系统和游戏逻辑

## 功能说明

### 游戏命令

- `恶魔轮盘.创建游戏` - 创建一个新游戏
- `恶魔轮盘.加入游戏` - 加入已有的游戏
- `恶魔轮盘.开始游戏` - 开始游戏（仅玩家1）
- `恶魔轮盘.对战信息` - 查看当前对战信息
- `恶魔轮盘.道具说明 [道具名]` - 查看道具说明
- `恶魔轮盘.结束游戏` - 结束游戏（玩家或管理员）

### 游戏内操作

- 发送道具名称使用道具
- 发送"自己"或"对方"开枪

### 道具列表

1. **手锯** - 下一发造成双倍伤害，不可叠加
2. **放大镜** - 查看当前膛内的子弹
3. **啤酒** - 卸下当前膛内的子弹
4. **香烟** - 恢复1点生命值
5. **手铐** - 跳过对方的下一回合
6. **过期药物** - 50%概率恢复2点生命值，50%概率损失1点生命值
7. **逆转器** - 转换膛内的子弹，实弹变为空包弹，反之亦然
8. **骰子** - 掷一个六面骰子，根据点数触发不同的效果

### 管理员权限

- Bot管理员可以强制结束任何游戏
- 使用 `恶魔轮盘.结束游戏` 命令

## 注册信息

- **包名**: bsr (buckshot roulette 的缩写)

## 并发安全

- 使用 `sync.RWMutex` 保护游戏状态
- 每个游戏实例都有独立的互斥锁
- 所有游戏状态修改都在锁保护下进行
